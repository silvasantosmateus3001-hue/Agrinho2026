# 🌾 Fazenda do Futuro

**Simulador Web Interativo** de uma fazenda sustentável do ano de 2047.

Um projeto educativo e divertido que simula o gerenciamento de uma fazenda futurista usando tecnologias avançadas como edição genética, hidroponia, robótica e inteligência artificial.

## 🚀 Como Executar

### Opção 1: Mais Fácil (Recomendada)

1. Baixe os 3 arquivos:
   - `index.html`
   - `styles.css`
   - `script.js`

2. Coloque os três arquivos na **mesma pasta**

3. Abra o arquivo `index.html` no seu navegador (duplo clique)

Pronto! O simulador já está funcionando.

### Opção 2: Usando Live Server (VS Code)

```bash
